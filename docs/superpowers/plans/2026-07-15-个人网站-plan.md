# 个人网站 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个面向求职的个人展示网站，包含技能图谱、实习经历、6个项目、专利与教育、简历下载共6个页面。

**Architecture:** Vite + React 18 + TypeScript + React Router v6 多页 SPA，Tailwind CSS 样式，数据集中在 `src/data/content.ts` 管理，Brittany Chiang 深海军蓝+青绿配色。

**Tech Stack:** Vite 6, React 18, TypeScript 5, React Router v6, Tailwind CSS 3, @tsparticles/react

## Global Constraints

- 所有内容中文，技术名词保留英文
- 配色：Navy `#0a192f` / Light Navy `#112240` / Lightest Navy `#233554` / Slate `#8892b0` / Light Slate `#a8b2d1` / Lightest Slate `#ccd6f6` / White `#e6f1ff` / Green `#64ffda`
- 字体：Playfair Display（hero/page titles 衬线体克制使用）、Inter（正文）、Fira Code（代码/标签）
- 导航标签无编号，纯文字；板块标题无编号前缀
- 卡片圆角 `rounded-sm`（近直角，terminal/IDE 风格）
- 卡片 hover 上浮 4px + 青绿阴影，transition 200ms
- 页面切换 fade-in 200ms
- 导航栏固定顶部，毛玻璃效果 `backdrop-blur bg-navy/80`
- 粒子背景仅首页，极低密度（15 粒子），点缀氛围
- 首页 Hero 采用"终端会话"签名风格（`$ whoami` 等命令行形式）
- 无占位数据，所有内容直接从简历填充
