## 一、技术选型

| 项目 | 选择 |
|------|------|

## 三、视觉语言

- **卡片**：`bg-light-navy` 填充，1px 细边框（Lightest Navy），hover 上浮 + 青绿微阴影
- **编号**：各板块标题前加数字编号（如 `01.`），青绿色 + Fira Code 字体
- **链接/按钮**：青绿色 `#64ffda`，hover 时加微弱光晕
- **粒子背景**：仅首页 hero 区使用 `@tsparticles/react`
- **导航栏**：固定顶部，毛玻璃效果（`backdrop-blur`），logo 左 + 导航链接右

## 七、动效规范

| 元素 | 动效 | 参数 |
|------|------|------|
| 首页标题 | 淡入上移 | `opacity 0→1, translateY 20px→0`，延迟 200ms |
| 页面切换 | 淡入 | 页面级 fade-in，200ms |
| 卡片 hover | 上浮 + 阴影 | `translateY -4px` + 青绿色阴影，transition 200ms |
| 技术标签 hover | 变色 | `color` 亮度提升 |
| 导航栏 | 毛玻璃 | `backdrop-blur-sm bg-navy/80` |
| 粒子背景 | 缓慢漂浮 | `@tsparticles/react`，低密度，仅在首页 |

无花哨过度动画，保持克制和专业感。
